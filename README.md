# Angular Calculator

Calculadora desenvolvida com **Angular 21** com o objetivo de praticar conceitos modernos do framework, principalmente o gerenciamento de estado reativo utilizando **Angular Signals**. Durante o desenvolvimento, foram aplicados recursos como componentes standalone, comunicação entre componentes por meio de serviços, criação de estados derivados com `computed` e execução de efeitos reativos com `effect`.

O projeto está disponível para visualização aqui:
**[Angular Calculator](https://givanyo-angular-calculator.vercel.app/)**
